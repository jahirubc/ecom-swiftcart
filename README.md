1) What is the difference between null and undefined?
   Null indicates absence of value where undefined means no value assinged on a declared variable.
2) What is the use of the map() function in JavaScript? How is it different from forEach()?
   Map() is used for creating new array after transforming each element of the existing array. forEach() perform iteration on each element but do not create new array.
3) What is the difference between == and ===?
   == comapare the value only where === compare both value and type.
4) What is the significance of async/await in fetching API data?
   To make asynchronous code look synchronous.
5) Explain the concept of Scope in JavaScript (Global, Function, Block).
Global scope: Accessible anywhere in the program.
Function scope: Accessible only inside the function where it’s declared.
Block scope: Accessible only inside the block {} it’s declared.
